Page for data on League of Legends

To help analyze biometric differences in skill level, we employed various machine learning models to classify participants based on heart rate and movement during specific gameplay events. We trained Long Short-term Memory (LSTM), Random Forest, and Suppert Vector Machine (SVM) classifiers on features that included participant heart rate, base heart rate, actions per minute, and key event labels. By comparing model performance, we identified a correlation between biometric indicators and player skill level. The LSTM model was particularly useful for capturing temporal patterns in the features, offering insight into how players respond to different in-game situations. Overall, the results suggest that biometric data can serve as an indicator of skill level in video game players and provide a foundation for future research in this area.
